# Dijkstra 알고리즘

최단 거리가 갱신 되었다는 것은 경유지가 바뀌었다는 뜻

시작 정점 A의 경우 A->A->K(다음 정점) 으로 생각함

```mermaid
graph LR;
  A((a))--3-->B((b));
  A--5-->C((c));
  B--2-->C;
  C--1-->B;
  B--6-->D((d));
  C--4-->D;
  C--6-->E((e));
  D--2-->E;
  D--3-->F((f));
  E--6-->F;
```
